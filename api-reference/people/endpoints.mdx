## Overview

The People API allows you to manage users, stakeholders, and team members within your Web3 GRC ecosystem.

<Note>
  All People API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/people
