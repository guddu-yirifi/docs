## Overview

The News API allows you to manage news articles, announcements, and updates within your Web3 GRC system.

<Note>
  All News API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/news
