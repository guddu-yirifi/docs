## Overview

The Feature API allows you to manage system features, capabilities, and feature toggles within your Web3 GRC platform.

<Note>
  All Feature API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/feature
