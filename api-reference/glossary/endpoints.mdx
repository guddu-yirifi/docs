## Overview

The Glossary API allows you to manage terminology, definitions, and educational content within your Web3 GRC system.

<Note>
  All Glossary API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/glossary
