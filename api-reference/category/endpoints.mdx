## Overview

The Category API allows you to manage categories within your system. You can create, read, update, and delete categories.

<Note>
  All Category API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/category
