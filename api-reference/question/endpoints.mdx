## Overview

The Question API allows you to manage questions, answers, and Q&A content within your Web3 GRC knowledge base.

<Note>
  All Question API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/question
