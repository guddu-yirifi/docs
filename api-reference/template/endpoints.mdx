## Overview

The Template API allows you to manage reusable templates for documents, workflows, and compliance processes within your Web3 GRC system.

<Note>
  All Template API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/template
