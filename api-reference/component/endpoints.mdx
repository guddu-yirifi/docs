## Overview

The Component API allows you to manage components within your Web3 GRC system. Components represent modular parts of your governance, risk, and compliance infrastructure.

<Note>
  All Component API endpoints require authentication via Bearer token.
</Note>

## Authentication

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dev-api.yirifi.ai/api/client/web3grc/api/component
