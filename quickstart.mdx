# Quickstart Guide

Get up and running with the Yirifi.ai API in just a few steps. This guide will walk you through making your first API call and exploring key features.

## Get your API key

Before you can start using the Yirifi.ai API, you'll need to get your API key.

<Steps>
  <Step title="Sign up">
    Create your free account at [yirifi.ai](https://yirifi.ai)
  </Step>
  <Step title="Get your API key">
    Navigate to your [dashboard](https://yirifi.ai/dashboard) and generate your API key
  </Step>
  <Step title="Save your key">
    Store your API key securely - you'll need it for all requests
  </Step>
</Steps>

<Warning>
Keep your API key secure and never share it publicly or commit it to version control.
</Warning>

## Make your first request

Let's start by fetching all categories to see what's available in your Yirifi.ai account.

<CodeGroup>
```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.yirifi.ai/v1/category"
```

```javascript JavaScript
const response = await fetch('https://api.yirifi.ai/v1/category', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const categories = await response.json();
console.log(categories);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.yirifi.ai/v1/category', headers=headers)
categories = response.json()
print(categories)
```
</CodeGroup>

## Create your first category

Now let's create a new category to organize your content.

<CodeGroup>
```bash cURL
curl -X POST "https://api.yirifi.ai/v1/category" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My First Category",
    "description": "A category created via the API"
  }'
```

```javascript JavaScript
const newCategory = await fetch('https://api.yirifi.ai/v1/category', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My First Category',
    description: 'A category created via the API'
  })
});

const category = await newCategory.json();
console.log('Created category:', category);
```

```python Python
import requests

data = {
    'name': 'My First Category',
    'description': 'A category created via the API'
}

response = requests.post(
    'https://api.yirifi.ai/v1/category',
    headers=headers,
    json=data
)

category = response.json()
print('Created category:', category)
```
</CodeGroup>

<ResponseExample>
```json Response
{
  "id": "cat_125",
  "name": "My First Category",
  "description": "A category created via the API",
  "createdAt": "2024-01-17T09:15:00Z"
}
```
</ResponseExample>

<Check>
**Success!** You've successfully created your first category using the Yirifi.ai API.
</Check>
